Lil

A command line utility for mobile web developers that will redirect a subdomain of lilup.co to the IP address of the machine you run it on.

Why Is That Useful?

It saves typing in your phone's browser each time the server's DHCP lease renews.

Okay, I'm In

  1. curl -O http://lilup.co/lil && chmod +x lil && ./lil my-app 8080
  2. http://my-app.lilup.co will then redirect to your local dev server.
  3. When your IP address changes, run lil <name> [<port>] again.
